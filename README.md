## Sistemas de Big Data: Actividad 2.5. Perceptrón multicapa

### Objetivos

1. Implementar un flujo de trabajo de Deep Learning: Desde la ingesta de datos hasta la predicción final.
2. Manejo de Librerías vs. Algoritmia Pura: Ser capaz de utilizar frameworks de alto nivel o, alternativamente, demostrar una comprensión profunda implementando el algoritmo desde cero.
3. Optimización Sistemática: Aplicar técnicas de Grid Search para evitar el ajuste manual y aleatorio de hiperparámetros.
4. Evaluación Crítica: Interpretar métricas más allá del accuracy global, analizando errores específicos mediante matrices de confusión.
5. Documentación Profesional: Sintetizar decisiones técnicas y resultados en una memoria formal.
